# Devok Monitor — Electron App

## Como usar

### Opção 1: Baixar o .exe (recomendado)
1. Acesse SisFin → Configurações → Devok Monitor
2. Clique "Baixar App"
3. Execute o instalador
4. Abra o app

### Opção 2: Rodar do código
```bash
npm install
npm start
```

## Funcionalidades
- Selecionar pasta do Devok
- Monitoramento a cada 30 segundos
- Agendamento diário em horário definido
- Verificação manual
- Log completo
- Funciona após standby do PC

## Gerar .exe
```bash
npm install
npm run build
```
O .exe será gerado na pasta `dist/`.
