# Devok Monitor — Importador Automatico de XMLs

## Como usar

### Opcao 1: Rode direto (requer Python)
1. Baixe `devok_monitor.py`
2. Clique duplo ou rode: `python devok_monitor.py`
3. O app detecta e instala dependencias automaticamente

### Opcao 2: Instalador completo (Windows)
1. Baixe `DevokMonitor.bat`
2. Coloque na mesma pasta que `devok_monitor.py`
3. Clique duplo no `.bat`
4. Ele instala Python e dependencias automaticamente
5. O app abre sozinho

## Funcionalidades

- Selecionar pasta do Devok
- Monitoramento a cada 30 segundos
- Agendamento diario em horario definido
- Verificacao manual (botao)
- Log completo
- Statisticas de importacao

## Configuracao

O app salva a configuracao em `config.json` na mesma pasta.

## Requisitos

- Windows 10/11
- Python 3.8+ (instalado automaticamente pelo .bat)
